# Monitor Zasobow
Program napisany w C++ i uruchamiany w Linux'ie (Ubuntu) do wyswietlania i monitorowania zasobow komputera takich jak
zuzycie pamieci, statyczne informacje na temat CPU (model, liczba rdzeni), dynamiczne informacje na temat CPU (zuzycie w czasie rzeczywistym) oraz podglad aktywnych procesow
